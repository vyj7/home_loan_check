# Home Loan Check

## Overview

Home Loan Check is a web-based application that helps users analyze home loan options and make informed financial decisions. The application provides comprehensive loan analysis including EMI calculations, tax benefits, affordability checks, and investment comparisons.

## Features

- **EMI Calculator**: Calculate Equated Monthly Installments based on loan amount, interest rate, and tenure
- **Tax Benefits Analysis**: Understand tax deductions available under Section 24(b) and Section 80C
- **Affordability Check**: Determine loan eligibility based on income and expenses
- **Investment Comparison**: Compare loan options with investment alternatives
- **Amortization Schedule**: View detailed breakdown of principal and interest payments over time

## Usage

The application is a static web application that runs entirely in the browser. Simply open `index.html` in a web browser to use the calculator.

### Key Inputs

- Monthly Income
- Property Value
- Interest Rate
- Loan Tenure
- Down Payment
- Tax Regime (Old vs New)

### Outputs

- EMI Amount
- Total Interest Payable
- Tax Savings
- Affordability Analysis
- Amortization Schedule

## Files

- `index.html` - Main application file
- `check.html` - Additional check functionality
- `inv.html`, `inv2.html` - Investment comparison views
- `pck.html` - Package/portfolio view

## Technology

This is a client-side application built with:
- HTML5
- CSS3
- Vanilla JavaScript

No backend server or database is required.
